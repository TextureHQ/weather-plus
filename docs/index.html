Weather Plus

Typed weather data from multiple providers, one API.

Weather Plus wraps NWS, OpenWeather, Tomorrow.io, and Weatherbit behind a single, type-safe interface with built-in caching and graceful fallback.

View on npm GitHub repository

Why Weather Plus?

Multi-provider fallback

Supply an ordered list of providers. Weather Plus will try each one in turn and stop as soon as a provider succeeds.

Strict TypeScript types

All responses conform to a shared schema with standardized weather conditions to keep your domain logic predictable.

Smart caching

Use an in-memory cache out of the box or plug in Redis for multi-instance deployments. Geohash keys keep stores compact.

Observability hooks

Outcome reporters track provider successes, failures, and latency so you can route alerts or metrics wherever you like.

Quick start

yarn add weather-plus
import WeatherPlus from 'weather-plus';

const weather = new WeatherPlus({
  providers: ['weatherbit', 'openweather', 'nws'],
  apiKeys: {
    weatherbit: process.env.WEATHERBIT_API_KEY!,
    openweather: process.env.OPENWEATHER_API_KEY!,
  },
});

const report = await weather.getWeather(40.7128, -74.0060);
console.log(report.conditions.value);

Configuration overview

Provider ordering

`providers` accepts an array like `['weatherbit', 'openweather', 'nws']`. The first provider to resolve wins; failures automatically fall through to the next provider.

API keys

Use the `apiKeys` map to supply credentials keyed by provider name. Providers without keys (e.g. NWS) can be omitted.

Caching

Pass a Redis client to enable cross-instance caching. Otherwise an in-memory store is used. Adjust TTL and geohash precision as needed.

Observability

Inject a custom outcome reporter to log or emit metrics whenever a provider call succeeds or fails.

Fallback & error handling

Weather Plus normalizes provider errors into a shared taxonomy so you can route retries and observability in a single place. When all providers fail, the most recent error is re-thrown with cause information preserved.
